membuat algoritma untuk memilih angka terbesar dari 3 angka, dengan angka yg berbeda

desc
1. input num1, num2, num3
2. Declar max is 0
3. if num1 is greater than equals num2 and num 1 is greater than equals num3, set the max is num1
4. Otherwise if num is greater than equals num1 and num2 is greater than equals num2, set the max is num2,
5. otherwise set the max is num3


input num1,num2, num3
declare max,-0

if (num1>=num2&& num1>=num3) then
    max<-num1

else if (num2>=num1 && num2>= num3) then
    max <- num3

print max

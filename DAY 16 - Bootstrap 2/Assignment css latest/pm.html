Bootcamp

Pelajari Agility Dalam Membangun Digital Product yang Sukses Menjawab Kebutuhan Customer!

Bootcamp Product Management merupakan program pelatihan intensif yang bertujuan untuk meningkatkan kemampuan peserta dalam melakukan ideation, market research, wireframing, prototyping, hingga menyusun user stories bersama praktisi digital berpengalaman selama 4,5 bulan tanpa memerlukan background IT.

    Apa yang Akan Kamu Dapatkan dari Program Bootcamp Ini?

  • Extra 20+ live session dan video learning untuk menunjang pembelajaran kelas
  • Practical exercises setiap pertemuan
  • Real problems based final project
  • Konsultasi pribadi bersama mentor ahli berpengalaman
  • Program graduation dan penyaluran kerja

Metode Belajar

Join WhatsApp Group

Sebelum kelas dimulai, kamu akan dihimbau untuk join grup WhatsApp terlebih dahulu untuk memudahkan berkoordinasi terkait pembelajaran bersama mentor.

Pra-Mentoring dengan Guide Book

Kamu akan mendapatkan file PDF, rangkuman materi yang akan dipelajari selama beberapa pertemuan ke depan.

Bebas Tanya Mentor

Kamu bisa bertanya dengan mentor saat sesi kelas maupun di luar kelas, lho!

Final Project

Diwajibkan untuk mempresentasikan hasil project yang telah dibuat sebagai syarat kelulusan program

Sharing Session

Kamu akan mendapatkan evaluasi sekaligus sharing session dengan mentor atas project yang telah dikerjakan. Pada sesi ini, kamu juga berkesempatan mengulas kembali semua materi yang sudah dipelajari bersama mentor.

Project Session

Kamu berkesempatan untuk mengerjakan project dengan topik yang akan ditentukan oleh mentor setiap minggunya.
